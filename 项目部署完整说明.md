## 🐙 GitHub上传详细步骤

### 第1步：创建仓库
```
访问: https://github.com/new
仓库名: mindmap-pro
说明: Professional MindMap with Theme System
选择: Public
```

### 第2步：生成Token
```
访问: https://github.com/settings/tokens
点击: Generate new token (classic)
名称: mindmap-pro-token
权限选择:
  ☑ repo (full control)
  ☑ workflow
有效期: 90 days
点击: Generate token
⚠️  复制Token（只显示一次！）
```

### 第3步：运行上传工具
```
双击: 上传到GitHub.bat
输入: your_github_username
输入: mindmap-pro
输入: 你复制的token
等待完成...
```

### 第4步：启用GitHub Pages（可选）
```
打开: https://github.com/你的用户名/mindmap-pro
点击: Settings
点击: Pages (左侧菜单)
Branch: main
Folder: / (root)
点击: Save

稍等1-2分钟...
访问: https://你的用户名.github.io/mindmap-pro
```

## 🆘 常见问题

### Q: 主题选择会不会丢失数据？
A: **不会！** 主题只改变视觉样式，完全不影响你的导图内容。

### Q: 切换主题的速度快吗？
A: **非常快！** <100ms响应时间，几乎感受不到延迟。

### Q: 主题选择会被保存吗？
A: **会！** 自动保存到浏览器，下次打开自动应用。

### Q: 可以离线使用吗？
A: **完全可以！** 所有功能都在本地运行，无需网络。

### Q: GitHub上传失败怎么办？
A: 检查以下几点:
- GitHub用户名是否正确
- Token是否复制完整
- 仓库是否已创建
- 网络连接是否正常
