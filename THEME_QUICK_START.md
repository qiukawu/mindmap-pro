## 🔧 进阶用法（可选）

### 通过浏览器控制台切换
按 `F12` 打开开发者工具，粘贴以下代码：

```javascript
// 切换到商务蓝
mindMap.setTheme('business');

// 切换到清新绿
mindMap.setTheme('fresh');

// 切换到暗黑模式
mindMap.setTheme('dark');

// 切换到多彩主题
mindMap.setTheme('colorful');

// 切换到极简白
mindMap.setTheme('minimal');
```

按 Enter 执行，主题会立即改变 ⚡

### 查看当前主题
```javascript
// 显示当前主题名称
console.log(mindMap.themes.currentTheme);

// 显示当前主题的完整信息
console.log(mindMap.themes.getCurrentTheme());
```

### 清除保存的主题（重置为默认）
```javascript
localStorage.removeItem('mindmap_theme');
location.reload(); // 刷新页面
```

## ❓ 常见问题（5秒回答）

**Q: 主题选择会影响我的笔记吗？**
A: 不会！只改变颜色，数据完全不变

**Q: 可以保存多个主题吗？**
A: 目前只能保存1个，但可以随时切换

**Q: 主题的颜色是怎么选的？**
A: 遵循专业配色原则，每个主题的颜色都经过精心设计

**Q: 暗黑模式真的能保护眼睛吗？**
A: 长期使用会减少眼睛疲劳，但最好还是定时休息

**Q: 可以自定义颜色吗？**
A: 目前不支持，但我们可以考虑在未来版本中添加

**Q: 在手机上能用主题吗？**
A: 可以的，主题在所有设备上都能用

**Q: 主题会自动同步到其他设备吗？**
A: 目前不会，因为保存在本地浏览器。未来可能支持云同步
