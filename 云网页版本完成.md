## 🚀 部署到 GitHub（3分钟快速版）

### 前置条件
- ✓ GitHub 账号（<a href="https://github.com/signup">注册</a>）
- ✓ Git 安装（<a href="https://git-scm.com">下载</a>）

### 命令行步骤

在 `C:\Users\Juan\mindmap-pro` 文件夹中打开 PowerShell，运行：

```powershell
# 1. 初始化Git仓库
git init

# 2. 添加所有文件
git add .

# 3. 创建首次提交
git commit -m "Initial commit: MindFlow Pro with professional theme system"

# 4. 重命名主分支
git branch -M main

# 5. 添加远程仓库（替换 你的用户名）
git remote add origin https://github.com/你的用户名/mindmap-pro.git

# 6. 推送到GitHub
git push -u origin main
```

### 启用 GitHub Pages

1. 访问 `https://github.com/你的用户名/mindmap-pro`
2. 点击 **Settings** → **Pages**
3. 选择：
   - Branch: **main**
   - Folder: **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟

### 完成！

访问：`https://你的用户名.github.io/mindmap-pro`
