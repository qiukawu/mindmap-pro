## ✅ 立即验证三大修复（3 分钟）

### 验证 1️⃣：侧栏展开按钮（关键）

```
1. 点击左侧栏的折叠箭头 →  左栏缩小

```

```
✅ 验证通过：能输入自定义模型名称
❌ 验证失败：查看浏览器控制台（F12）有无错误
```

```
1. 不配置 API（或清除之前的配置）
2. 新建一个节点，输入文本
3. 右键点击节点 → AI → 分析
4. 应该看到本地生成的建议

✅ 验证通过：无 API 也能使用 AI
❌ 验证失败：刷新页面或检查控制台错误
```

## 🔧 常见操作

### 配置 API（如果需要）
```
1. 点击 ⚙️ 设置
2. 填写 API 地址（例如：https://api.openai.com/v1/chat/completions）
3. 填写 API 密钥
4. 选择或输入模型名（例如：gpt-4 或 claude-3-opus）
5. 点击"测试连接"验证
6. 点击"保存"
7. 关闭对话框，AI 功能自动使用你的 API
```

### 导出为图片
```
1. 点击工具栏的"导出"
2. 选择 PNG 或 JPG
3. 等待生成
4. 浏览器会下载高清图片
```

### 导出并分享
```
1. 导出为 JSON（完整数据）
2. 发送给他人
3. 他人打开应用 → 导入 → 选择你的 JSON 文件
4. 完整的思维导图恢复
```

## 🐛 遇到问题？

### 问题 1：侧栏按钮看不见
**解决**：
1. 点击工具栏刷新按钮（或 F5 刷新）
2. 清除浏览器缓存：Ctrl+Shift+Del
3. 刷新页面

### 问题 2：API 设置无法保存
**解决**：
1. 检查 API 地址是否完整（包含 `/v1/chat/completions` 等）
2. 检查 API 密钥是否正确
3. 打开控制台（F12）查看错误信息

### 问题 3：AI 功能无反应
**解决**：
1. 检查是否正确配置了 API
2. 点击"测试连接"验证
3. 如果不想用 API，本地模式也能工作

### 问题 4：数据丢失
**解决**：
1. 检查浏览器是否启用了 LocalStorage
2. 打开浏览器控制台：`localStorage.getItem('mindmap_data')`
3. 如果为空，可从备份导入

### 问题 5：页面很卡
**解决**：
1. 检查节点数量（超过 500 个会变慢）
2. 删除不需要的节点
3. 刷新浏览器
4. 分成多个较小的思维导图
